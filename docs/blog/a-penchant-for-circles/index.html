A Penchant for Circles

Published on 2006-07-11 21:34:00+00:00
Fun  

Apparently I like Circles, as is evidenced by my latest desktop background. If you've looked through my other background images, you'll notice that I also like a simple, unobtrusive image behind my work.

After I created this one, it seemed to remind me of something... and after a little though, I remembered Sacrifice, originally developed by Shiny. I won't go into a long spiel, but it had beautiful graphics, a very interesting interface (once you got the hang of it), and quite possible some of the best AI that I've seen in a game for a while... not that I've actually played any games in a while.

It's just too bad I'm the only person I know that played the game... it was nice.
